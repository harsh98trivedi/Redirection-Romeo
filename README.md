# Redirection Romeo

![Redirection Romeo Banner](assets/images/meta.jpg)

**Redirection Romeo** is a modern, lightweight WordPress plugin designed to make redirect management effortless and beautiful. Say goodbye to clunky tables and outdated interfaces; welcome to a sleek, card-based dashboard that supports the latest SEO standards, including **308 Permanent Redirects**.

-   **Modern UI/UX**: Built with a clean, card-based interface that feels like a modern SaaS app, not a legacy plugin.
-   **Full Standard Support**: Supports **301**, **302**, **307**, and the modern **308 (Permanent Redirect, Preserve Method)** status codes.
-   **Instant Search**: Real-time, optimistic search filtering to find your redirects instantly.
-   **Internal Linking**: Intelligent autocomplete to easily find and redirect to your internal Posts and Pages.
-   **Performance Focused**: Lightweight and fast, using native JavaScript with zero heavy dependencies.
-   **Hit Counting**: Track the popularity of your links with built-in hit counters.

## ğŸ› ï¸ Usage

### Creating a Redirect
1.  Click the **"Create New Redirect"** button.
2.  Enter your desired **Source Slug** (e.g., `my-offer`).
3.  Select the **Target Type**:
    *   **External URL**: Enter any web address (e.g., `https://google.com`).
    *   **Internal Post**: Type to search for any page or post on your site.
4.  Choose your **HTTP Code** (Recommend `308` for modern permanent redirects).
5.  Click **Save**.

### Managing Redirects
*   **Search**: Use the rounded search bar to filter redirects by slug or target.
*   **Edit**: Hover over any card and click the <span style="color:#64748b">âœ</span> Edit icon to modify it.
